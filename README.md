- Responsive
- Dark - Light mode

## How to use

```
git clone https://github.com/Ashish-CodeJourney/Ashish-CodeJourney.github.io.git
```
```
cd Ashish-CodeJourney.github.io
```
```
npm install
```
```
npm run dev
```

Edit ```data.json``` file on `src/assets/data.json` and put your own data
